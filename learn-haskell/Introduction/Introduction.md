# Introduction

## So What's Haskell?
Haskell is a :-
* Purely functional programming language
* Uses lazy evaluation by default
* Statically typed (with type inference)

## ghci
ghci commands: -

```
:l <script>.hs
```
Load the functions defined in `<script>.hs`.

```
:r 
```
reload the current script.

```
:t <expr>
```
show the type of an expression
